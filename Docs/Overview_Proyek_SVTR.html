Overview Proyek SVTR

Scene Text Recognition pada Kemasan Pangan — bahan belajar

Sistem yang membaca tanggal kadaluarsa & kode produksi pada kemasan pangan secara otomatis dari foto, memakai Deep Learning SVTR + mekanisme confidence-aware (menolak hasil ragu, minta foto ulang).

1. Masalah & Solusi

Masalah: OCR biasa gagal baca kemasan karena permukaan mengilap (glare), cetakan dot-matrix kecil, permukaan melengkung, pencahayaan rendah/blur.

Solusi: Model SVTR di-fine-tune khusus teks kemasan + confidence-aware OCR.

Target: Akurasi >85% (normal), >75% (degradasi), CER/WER <15%, inferensi <10 detik, terintegrasi ke Android.

6. Pelajaran Penting

1. Konsistensi versi krusial. Model dilatih PaddleOCR 2.7, harus di-inference versi sama. Beda versi = output rusak.
2. Data leakage = metrik palsu. v1 tampak 91% tapi nyatanya jelek — 60% val "bocor" dari train (menghafal, bukan belajar). Setelah bersih: ~42-66% (jujur).
3. Dictionary harus sinkron. Model keluarkan index angka → dipetakan ke karakter via dict. Geser 1 = semua huruf salah.
4. Sintetik + riil saling melengkapi. Sintetik untuk variasi pola, riil untuk kondisi nyata (glare, dot-matrix).
5. Komposisi data menentukan fokus. Terlalu banyak teks umum membuat model terdistraksi dari target (kode/tanggal).

Bahan belajar proyek SVTR — disusun berdasarkan kondisi nyata pengembangan.
